Logratio plots

Besides ternary diagrams, logratio plots are a second way to visualise (U-Th)/He data on a two-dimensional surface. They plot log(Th/He) vs. log(U/He). The default scale goes from -1 to 6 on both axis, but can be automatically set to display the sample data at optimal resolution by clicking Options → Settings, or to any other values.
