Saving the data

The input data can be saved to a comma separated variable (.csv) file by clicking File → Save data. The U, std(U),Th, std(Th), He, and std(He) columns are mandatory and must be completed, the other columns are optional and may be left blank. To generate a complete dataset, including both Sm and the optional colour code, all columns must be completed before saving, for both the (U-Th)/He and (U-Th-Sm)/He options under Options → Input.
