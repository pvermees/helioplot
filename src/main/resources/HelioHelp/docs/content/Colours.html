Colours

The age contours of both ternary diagrams and logratio plots can be changed by clicking Options → Colours → Contour Colours. This brings up a colour scale, the minimum and maximum values of which can be selected by clicking the appropriate swatches or adjusting the RGB sliders.

A separate set of colours is used for the error ellipses of the (U-Th)/He data. This colours are either taken from the optional ninth column of the input table, or from the Sm concentrations, if the ninth column is empty. The ninth input column can be seen and edited only if the (U-Th)/He option is selected in Options → Input. Although this will hide the Sm content from the input table, the Sm values will still be used behind the scenes for the age calculation. Conversely, selecting the (U-Th-Sm)/He option in Options → Input reveals the Sm values and hides the colour data, but these will still be used to determine the fill colour of the ellipses and the associated colour legend. The colour scale can be changed by clicking Options → Colours → Ellipse Colours.
