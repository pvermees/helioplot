Importing data

Data files can be loaded into HelioPlot by clicking File → Open. Input files can be generated using HelioPlot's built-in editing capabilities, but can also be edited in other applications such as Microsoft Excel or any text editor as long as the following comma-separated format is used: 
samplename
U(1),σU(1),Th(1),σTh(1),Sm(1),σSm(1),He(1),σHe(1),C(1)
U(2),σU(2),Th(2),σTh(2),Sm(2),σSm(2),He(2),σHe(2),C(2)
...
U(n),σU(n),Th(n),σTh(n),Sm(n),σSm(n),He(n),σHe(n),C(n)
Where samplename is an alphanumerical string, and all other parameters are decimal numbers. Missing data should be indicated by a non-numerical character such as a blank space or NAN.
